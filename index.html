A la Carte Menu

The App connects Catering and On-Campus students enabling them to order food, collect feedback and more.

"The Full Stack"

The Frontend for the Client Side App and the Admin Page is developed using AngularJS with Ionic Framework.

The App uses REST API with JSON Formatting to communicate with the Servers.

PostgreSQL Database is used to store data with pg_partman and pg_cron extensions.

The Backend Server is written in Golang with the gorilla mux support.

The Entire stack is hosted on Salesforce's Heroku.
